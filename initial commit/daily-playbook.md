## RICE Score Calculator (Use for every new idea)

$$\text{Score} = \frac{\text{Reach} \times \text{Impact} \times \text{Confidence}}{\text{Effort}}$$

### Power Law Scales
| Scale | Reach | Impact | Confidence |
|-------|-------|--------|------------|
| **Exponential** | 1, 10, 100, 1000+ | 1, 10, 100 | 10%, 50%, 80% |
| **Example** | People reached | Career change potential | Probability of success |

### Pre-Filter Question
**"Does this have 10x+ exponential potential?"**
- ‚úÖ Yes ‚Üí Calculate RICE
- ‚ùå No ‚Üí Delegate/Automate/Skip

## Decision Making Framework

### When You're Stuck
1. What's the simplest next step?
2. Can I ship something in <2 hours?
3. Who can unblock me in <10 minutes?
4. Should I pivot to next highest RICE item?

### When You Want to Add a Feature
1. Calculate RICE score
2. Is it higher than current top item?
3. If yes, re-prioritize
4. If no, add to backlog and forget

### When Something Isn't Working
```
IF confidence dropped >50% THEN
    Log decision ("Killed X because confidence collapsed")
    Archive work (might be useful later)
    Pivot to next highest RICE item
END IF
```

## The Kill Decision

### When to Kill a Project/Task
- Confidence dropped below 5%
- Better opportunity emerged (2x+ higher RICE)
- Runway getting critical
- You're avoiding it for >3 days

### How to Kill
1. Log in decisions table with full reasoning
2. Extract lessons (what assumption was wrong?)
3. Archive code/docs
4. Immediately start next highest RICE item
5. **Do not feel guilty** - killing bad bets is a skill

## Emergency Protocols

### Blocked >24 Hours
```
1. Write down the blocker in detail
2. Set 2-hour timer for fix attempt
3. If still blocked ‚Üí Pivot
4. Come back tomorrow with fresh eyes
```

### Interview Pipeline Dried Up
```
1. Double outreach velocity (10 DMs/day)
2. Add new job sources
3. Lower fit score threshold
4. Ask network directly for intros
```

### Feeling Burned Out
```
1. Reduce next sprint commitment 20%
2. Take full evening off (no laptop)
3. Review sustainability in retro
4. Remember: 8 weeks, not forever
```
